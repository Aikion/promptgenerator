# ✨ Prompt Generator — AiKiOn Akademie

Ein einfacher, browserbasierter Prompt Generator für KI-Tools.
Aus einer unstrukturierten Idee wird automatisch ein professioneller Prompt — ohne Programmierkenntnisse, ohne Login, ohne App-Installation.

---

## 🚀 Nutzung

Einfach den Link öffnen und loslegen:

1. **Anfrage eingeben** — beschreibe in eigenen Worten, was du möchtest
2. **Zweck wählen** — z.B. Bild, Video, Text, Code, E-Mail ...
3. **KI-Tool wählen** — z.B. ChatGPT, Midjourney, Claude ...
4. **Stil / Ton wählen** — passend zum gewünschten Ergebnis
5. **Erweiterte Optionen** — optional: Beleuchtung, Format, Zielgruppe u.v.m.
6. **„Prompt generieren"** klicken
7. **Kopieren & verwenden** — direkt in dein KI-Tool einfügen

## ⚙️ Technische Details

- **Reine HTML/CSS/JavaScript Datei** — kein Server, kein Backend, kein API-Key erforderlich
- **Funktioniert offline** — einmal geladen, kein Internet nötig
- **Regelbasierte Prompt-Generierung** — intelligente Templates je nach Zweck & Tool
- **Mobil-optimiert** — funktioniert auf Smartphone und Tablet
- **Ausgabeformate** — Markdown, JSON oder Fließtext wählbar

## 🔧 Lokal verwenden

Keine Installation nötig. Einfach die `index.html` Datei im Browser öffnen:

```
Doppelklick auf index.html → öffnet sich im Standard-Browser
```

---

## 🌐 Deployment via GitHub Pages

1. Repository auf GitHub als **Public** anlegen
2. `index.html` hochladen
3. Unter **Settings → Pages → Branch: main** aktivieren
4. Zugriff über: `https://USERNAME.github.io/promptgenerator`
